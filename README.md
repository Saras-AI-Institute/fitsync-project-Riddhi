# FitSync - Health Analytics Platform 🚀
*Empower Your Health Journey with Data-Driven Insights*

## Project Overview
FitSync is a sophisticated 3-page health analytics dashboard crafted with Python and Streamlit to aid individuals in tracking and improving their well-being. This platform features a Main page, a Dashboard showcasing KPIs and interactive charts, and a Trends page offering insightful histograms and analyses. Designed for clarity and ease of use, FitSync transforms personal health data into actionable insights, enabling users to visualize and optimize their fitness journeys. It caters to health enthusiasts eager to gain comprehensive insights and control over their wellness metrics.

## How to Run
1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/yourusername/FitSync.git
   ```
2. Open a new [GitHub Codespaces](https://github.com/features/codespaces).
3. In the terminal, start the Streamlit application by running:
   ```bash
   streamlit run main.py
   ```
